About SpamShield

SpamShield is a machine learning-powered web application designed to detect and filter out SMS spam. Built with Python, Scikit-learn, and Flask, our classifier uses a voting ensemble model to make accurate predictions in real-time.

How It Works

  • Incoming messages are preprocessed using tokenization, stemming, and stopword removal.
  • A TF-IDF vectorizer converts text into numerical features.
  • The final prediction is made using a Voting Classifier combining SVM, Naive Bayes, and Extra Trees models.

Features

  • Single message classification
  • Bulk CSV upload classification
  • Clean, responsive UI
  • Optional user login for personalization

Technologies Used

  • Python + Flask (Backend)
  • Scikit-learn + NLTK (ML & NLP)
  • TF-IDF Vectorization
  • HTML, CSS, JavaScript (Frontend, Dark Theme)

Training Details

The model was trained on a public SMS spam dataset containing 5,574 labeled messages. After thorough preprocessing and vectorization, the ensemble classifier achieved over 95% accuracy on the test set.

Limitations

  • Accuracy may drop for non-English, slang-heavy, or code-mixed messages.
  • SpamShield is intended for educational/demo use and is not production-hardened.

Ethical Use

SpamShield is built to show how AI can contribute to safer communication. Please ensure you do not upload confidential or private messages without proper consent.
